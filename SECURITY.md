# Security Policy

## Reporting a Vulnerability

If you believe you have found a security vulnerability in RDAPify, please report it privately.

- **Email**: security@rdapify.com

Please include:
- A clear description of the issue and impact
- Steps to reproduce (PoC if available)
- Affected versions/commit hash
- Any suggested mitigation

We aim to acknowledge reports within 72 hours.

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |
| < 0.1   | :x:                |

## Security Best Practices

When using RDAPify:
- Keep the package updated to the latest version
- Review security advisories regularly
- Report any suspicious behavior
- Follow secure coding practices

## Disclosure Policy

- We follow responsible disclosure
- Security fixes are prioritized
- CVEs are assigned when applicable
- Public disclosure after fix is available

## Contact

- **Security Team**: security@rdapify.com
- **Emergency**: emergency@rdapify.com
- **Response Time**: Within 72 hours
